	※ About :
		Subtitles Tool(c) SuWen Creations, SuWen All rights reserved.

	※ Description :
		This is a subtitle tool
		provide a quick and easy subtitle solution
		while recording video
		this editor can display subtitles
		explain the current situation
		recorded video
		you don't need to use post-production software to add subtitles
		Is a good helper for meetings, presentations, and message delivery.

	※ Quick Guide-Turn on the subtitle tool:
		01) This tool in the Unity toolbar "window/text tool/subtitle tool" to open the "SUBTITLES EDITOR"
		02) You can enter the subtitle you want to display in the subtitle editor window
		03) After completing the subtitle input, click the subtitle player button to open the "SUBTITLES PLAYER" window
		04) In the player window, pre-input subtitles can be played manually or automatically
		05) It can be used with recording software or other screen capture functions to add subtitles quickly and easily.

	※ Quick guide-change language:
		01) First open the "SUBTITLES EDITOR" (In the Unity Toolbar Window/SuWen Tool/Subtitle Tool)
		02) Select "Display/Languge/" in the toolbar above the "SUBTITLES EDITOR"
		03) Select a language in this menu to switch the language of the tool interface.

中文

	※ 關於 :
		字幕工具（c）SuWen 創建，SuWen保留所有權利。

	※ 說明 :
		這是個字幕工具
		提供快速簡便的字幕解決方案
		錄製影片時
		該編輯器可以顯示字幕
		解釋當前情況
		錄製完成的影片
		您不需要使用後期製作軟件來添加字幕
		是會議、演示和訊息傳遞的好幫手。

	※ 快速指南 - 開啟字幕工具 :
		01) 本工具位於Unity工具列 Window/SuWen Tool/Subtitles Tool 可以開啟 "字幕編輯器"
		02) 在字幕編輯器視窗中可以輸入想要顯示的字幕
		03) 完成字幕的輸入，按下字幕播放器按鈕，則可開啟字幕播放器
		04) 於播放器中，可以手動或自動播放事先輸入的字幕
		05) 此時可以搭配錄影軟體或其他擷取畫面的功能，快速簡單的加上字幕

	※ 快速指南 - 改變語言 :
		01) 首先先開啟 "字幕編輯器" (位於Unity工具列 Window/SuWen Tool/Subtitles Tool)
		02) 在 "字幕編輯器" 上方的工具列選擇 "Display/Languge/"
		03) 在該選單中選擇一個語言，即可切換本工具介面的語言
